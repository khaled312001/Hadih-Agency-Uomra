# تطبيق هدية - نظام العمرة الإلكتروني

## نظرة عامة
تطبيق هدية هو نظام إلكتروني متكامل يعمل كوسيط بين المستفيدين ومزودي الخدمة لأداء مناسك العمرة. يوفر النظام شفافية كاملة من خلال توثيق المناسك بالفيديو وإرسالها للمستفيدين.

## المميزات الرئيسية

### 🎯 للمستفيدين
- **إنشاء طلبات العمرة**: إمكانية طلب العمرة للمتوفين والمرضى وكبار السن والمعاقين
- **اختيار حزم متنوعة**: حزم عمرة مختلفة بأسعار مناسبة
- **متابعة الطلبات**: تتبع حالة الطلب من البداية حتى الإكمال
- **توثيق بالفيديو**: استلام فيديوهات توثق أداء المناسك
- **نظام مراسلات**: التواصل المباشر مع مقدمي الخدمة

### 👨‍💼 لمقدمي الخدمة
- **تسجيل كخدمة**: إمكانية التسجيل كمقدم خدمة مؤهل شرعياً
- **إدارة الطلبات**: عرض وإدارة الطلبات المكلفين بها
- **رفع الفيديوهات**: توثيق أداء المناسك بالفيديو
- **نظام تقييم**: الحصول على تقييمات من المستفيدين

### 🔧 للإدارة
- **لوحة تحكم شاملة**: إدارة كاملة للنظام
- **إدارة المستخدمين**: إدارة المستفيدين ومقدمي الخدمة
- **إدارة الطلبات**: متابعة وإدارة جميع الطلبات
- **إدارة الحزم**: إضافة وتعديل حزم العمرة
- **تقارير وإحصائيات**: إحصائيات مفصلة عن النظام

## التقنيات المستخدمة

### Backend
- **Laravel 11**: إطار عمل PHP متطور
- **MySQL**: قاعدة بيانات علائقية
- **Laravel Sanctum**: للمصادقة والهدية
- **Laravel UI**: واجهات المصادقة
- **Spatie Laravel Permission**: إدارة الصلاحيات

### Frontend
- **Bootstrap 5**: إطار عمل CSS متجاوب
- **Font Awesome**: مكتبة الأيقونات
- **Google Fonts (Cairo)**: خطوط عربية جميلة
- **JavaScript**: للتفاعل الديناميكي

## هيكل قاعدة البيانات

### الجداول الرئيسية
- **users**: المستخدمين (مستفيدين، مقدمي خدمة، إدارة)
- **umrah_packages**: حزم العمرة المتاحة
- **service_providers**: مقدمي الخدمة المؤهلين
- **orders**: طلبات العمرة
- **payments**: معلومات الدفع
- **messages**: نظام المراسلات
- **videos**: فيديوهات توثيق المناسك

## التثبيت والتشغيل

### المتطلبات
- PHP 8.1 أو أحدث
- Composer
- MySQL 5.7 أو أحدث
- Node.js (اختياري للتجميع)

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone [repository-url]
cd tafweed-app
```

2. **تثبيت التبعيات**
```bash
composer install
```

3. **إعداد البيئة**
```bash
cp .env.example .env
php artisan key:generate
```

4. **تكوين قاعدة البيانات**
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=tafweed_db
DB_USERNAME=root
DB_PASSWORD=
```

5. **تشغيل Migration**
```bash
php artisan migrate
```

6. **إضافة البيانات التجريبية**
```bash
php artisan db:seed --class=UmrahPackageSeeder
php artisan db:seed --class=AdminUserSeeder
```

7. **تشغيل الخادم**
```bash
php artisan serve
```

## الصفحات الرئيسية

### الصفحات العامة
- **الصفحة الرئيسية** (`/`): صفحة الهبوط مع عرض الحزم والإحصائيات
- **تسجيل الدخول** (`/login`): صفحة تسجيل الدخول
- **إنشاء حساب** (`/register`): صفحة التسجيل

### صفحات المستخدمين
- **لوحة التحكم** (`/home`): لوحة تحكم المستخدم
- **الطلبات** (`/orders`): إدارة طلبات العمرة
- **إنشاء طلب** (`/orders/create`): إنشاء طلب عمرة جديد

### صفحات الإدارة
- **لوحة الإدارة** (`/admin/dashboard`): لوحة تحكم الإدارة
- **إدارة الحزم** (`/admin/packages`): إدارة حزم العمرة
- **إدارة الطلبات** (`/admin/orders`): إدارة جميع الطلبات
- **إدارة المستخدمين** (`/admin/users`): إدارة المستخدمين

## الأمان

### المصادقة والهدية
- نظام مصادقة متقدم باستخدام Laravel Sanctum
- هدية متدرج (مستفيد، مقدم خدمة، إدارة)
- حماية CSRF لجميع النماذج
- تشفير كلمات المرور

### حماية البيانات
- تشفير البيانات الحساسة
- حماية من SQL Injection
- حماية من XSS
- التحقق من صحة البيانات
